프로젝트 5

직원 출퇴근 기록 시스템

인증된 REST API와 중앙 게이트웨이를 통해 직원 출퇴근 데이터를 안전하게 관리하는 마이크로서비스 기반 시스템입니다.


이 프로젝트는 Spring Boot, Angular, Docker를 기반으로 한 **풀스택 마이크로서비스 애플리케이션**입니다. **JWT 기반 인증 시스템**을 Spring Cloud Gateway에서 적용하여 모든 출입 기록을 안전하게 처리하며, **중앙 API 라우팅 및 무상태 인증(stateless authentication)**을 통해 보안성과 확장성, 유지보수성을 동시에 확보했습니다.

아키텍처 (Architecture)

기술 스택 (Tech Stack)

  • Angular: 반응형 폼 및 BehaviorSubject 기반 실시간 UI 구현
  • Spring Boot: 직원 정보 및 출입 기록 관리를 위한 백엔드 REST API 마이크로서비스
  • Spring Security + JWT: JWT를 이용한 무상태 인증 및 역할 기반 접근 제어
  • Spring Cloud Gateway: API Gateway 역할, 서비스 간 라우팅 및 인증 필터 처리
  • Apache Kafka: 비동기 이벤트 처리를 위한 메시지 브로커
  • MongoDB: 로그 및 이벤트 스트림 저장용 NoSQL 데이터베이스
  • MySQL: 직원 프로필 및 출퇴근 기록용 관계형 데이터베이스
  • Docker: Docker Compose를 이용한 전체 서비스 컨테이너화 및 배포

✨ 주요 기능

  • JWT 토큰 기반의 안전한 로그인 및 역할별 접근 제어
  • Spring Cloud Gateway를 통한 중앙 API 인증 및 라우팅
  • 실시간 출근/퇴근 체크 및 타임스탬프 기록 기능
  • 유지보수가 쉬운 모듈화된 마이크로서비스 아키텍처

구성 구조 (Structure)

Spring Cloud Gateway

  • 모든 백엔드 마이크로서비스로 향하는 단일 진입점 역할 수행
  • 요청 라우팅 및 인증 필터 기능 처리

JWT 인증

  • 모든 API 요청은 JWT 기반의 무상태 인증으로 보호됨
  • 게이트웨이는 요청을 내부 서비스로 전달하기 전에 토큰을 검증함

Kafka

  • 서비스 간 메시지를 비동기적으로 중계하는 메시지 브로커 역할
  • 마이크로서비스 간 메시지 전달 처리
  • 기술: Apache Kafka, Kafka Topic, Spring Kafka (Producer/Consumer), kafka-python, Docker Compose

마이크로서비스

  • Attendance Service: 출퇴근 시간 기록 처리
  • Employee Service: 직원 등록, 로그인 및 역할 기반 접근 제어
  • Admin Dashboard: 직원 출퇴근 기록을 관리자 페이지에서 관리

데이터 저장 (Persistence)

  • Spring Data JPA와 MySQL을 사용하여 직원 정보 및 활동 내역 저장

컨테이너화 (Containerization)

  • Docker 및 Docker Compose를 활용한 전체 서비스 컨테이너 구성 및 배포
