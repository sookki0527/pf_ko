프로젝트 1

최신 뉴스 요약 및 질문 생성기

이 프로젝트는 ElasticSearch와 LangChain 기반의 RAG 파이프라인을 구축하여 최신 문서를 연동하고, 제공된 컨텍스트에 근거한 질문 생성과 정확한 답변을 제공하는 시스템을 구현했습니다.


이 프로젝트는 RAG(Retrieval-Augmented Generation) 아키텍처로, ElasticSearch에 최신 문서를 색인하고 LangChain 리트리버로 관련 컨텍스트를 수집한 뒤 LLM이 컨텍스트 기반으로만 답변하도록 설계되었습니다

기술 스택 (Tech Stack)

  • RAG: BM25 + (옵션) 벡터 검색, 근거 문서 인용
  • LangChain: Text Splitter(RecursiveCharacterTextSplitter), MultiQuery Retriever, ChatPromptTemplate
  • Flask (Python): REST/SSE API, CORS, (운영) gunicorn/waitress
  • Elastic Cloud (Elasticsearch): 단일 인덱스, sparse/dense 필드 매핑
  • Selenium: 헤드리스 크롤링, 정제/중복 제거

구조 (Structure)

데이터 크롤링 및 JSON 문서 매핑

  • Selenium으로 최신 뉴스/문서 크롤링 → 본문/제목/URL/작성일 추출
  • HTML 제거·공백 정리 후 JSON 스키마에 매핑

LangChain을 이용한 토큰화 및 텍스트 분할

  • RecursiveCharacterTextSplitter (예: chunk_size=800, chunk_overlap=120)
  • 메타데이터 유지: source, url, published_at 등

Elasticsearch 적재 및 인덱싱

  • 인덱스: rag-docs
  • 필드: content(text), source(keyword), url(keyword), published_at(date), embedding(dense_vector)
  • 업서트로 중복 방지, 갱신일 기준 재색인(신규 문서 빠른 반영)

GPT 모델 연동 및 프롬프트/질문

  • 모델: gpt-4o-mini
  • 시스템 프롬프트: “주어진 컨텍스트 내에서만 답변, 근거 없으면 ‘모르겠다’.”
  • 질문 예시: “텍스트 요약”, “저자는 어느 학교 교수인가?”, “핵심 수치만 뽑아줘”

Flask

  • 프론트엔드와 연결하기 위한 Python 백엔드 API (예: POST /api/ask, GET /api/stream for SSE)
